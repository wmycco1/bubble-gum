### Run Unit Tests

```bash
# All tests
npm test

# Unit tests only
npm run test:unit

# With coverage
npm run test:coverage

# Watch mode
npm run test:watch
```

### Run Integration Tests

```bash
# Integration tests
npm run test:integration

# With database
docker-compose up -d postgres redis
npm run test:integration
```

### Run E2E Tests

```bash
# All E2E tests
npm run test:e2e

# Specific browser
npm run test:e2e -- --project=chromium

# Headed mode
npm run test:e2e -- --headed

# Debug mode
npm run test:e2e -- --debug

# UI mode
npm run test:e2e:ui
```

### Run Performance Tests

```bash
# Lighthouse
npm run lighthouse

# K6 load test
k6 run load-test.js

# Bundle analyzer
ANALYZE=true npm run build
```

# npm audit

npm audit fix

# Snyk
